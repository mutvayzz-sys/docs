## System requirements

- **Windows:** Windows 10 or later (64-bit)
- **macOS:** macOS 10.14 (Mojave) or later (Intel and Apple Silicon)
- **Linux:** Ubuntu 18.04+, Fedora 30+, Debian 10+, or equivalent

These docs cover **Headmaster v0.2.1**.
